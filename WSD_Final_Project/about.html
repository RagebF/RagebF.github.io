About this project

A simple blog-style website built as a final project for a web development course.

Goal of the website

The main goal is to practice building a small but complete website using HTML, CSS, and Bootstrap 5. The site should be readable, responsive and accessible. It should also feel honest and personal, not like a template with empty buzzwords.

About the author

The blog is written by a student who is still learning. The focus is on explaining what was learned each week and what problems appeared along the way.

Topics covered

  • Basic HTML structure and semantic tags
  • Styling with CSS and Bootstrap 5 components
  • Responsive design and testing on different screen sizes
  • Accessibility, including alt text and clear labels

Tools used

The website is created with a normal code editor, the Chrome developer tools, and the Bootstrap documentation. Images are taken from free photo libraries such as Unsplash, Pexels, or Pixabay.
