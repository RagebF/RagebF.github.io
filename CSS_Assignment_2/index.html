About the Collection

Urban Moments is a small art gallery project created for web development studies. The photographs can be your own or royalty-free images. The layout highlights CSS positioning, color, and typography.

This website is built for educational purposes as part of the Web Site Development course at Laurea University of Applied Sciences.

Theme

Focus on modern city life: skylines, riversides, public transport, street details, and architecture.

Photography Tip

Try shooting the same street during the day and at night to compare how color and light transform the scene.
